# daily-quotes

A daily dose of motivational quotes.

![Animating quotes](./docs/quotes.gif)

![Quote admin](./docs/admin.png)

## Getting Started

Setup MySQL database:

```sh
cd server
mysql -u root < db.sql
```

Start the server:

```sh
cd server
npm install
npm start
```

Start the client:

```sh
cd client
npm install
npm start
```
